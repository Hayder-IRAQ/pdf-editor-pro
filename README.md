# 📄 PDF Editor Pro — محرر PDF احترافي

**Full-featured PDF editor built with Python — edit text, draw shapes, annotate, merge, watermark, and more.**

## ✨ Features

| Category | Features |
|---|---|
| ✏️ **Text Editing** | Add, edit, delete text — choose font size and color |
| 🎨 **Drawing Tools** | Rectangle, Circle, Line, Arrow, Freehand |
| 🖍️ **Annotations** | Highlight, Underline, Strikeout |
| 🖼️ **Images** | Insert images anywhere on any page |
| 📑 **Page Management** | Add, delete, rotate, extract pages |
| 🔗 **File Operations** | Merge multiple PDFs, extract pages to new file |
| 💧 **Watermark & Stamp** | Add watermarks and stamps (Draft, Confidential, etc.) |
| ✍️ **Signature** | Draw and insert handwritten signatures |
| 🔍 **Find & Replace** | Search and replace text across all pages |
| 📤 **Export** | Export pages as PNG images, extract text to TXT |
| ↩️ **Undo / Redo** | Full undo/redo history (up to 50 states) |
| 🌙 **Dark Theme** | Dark UI for comfortable editing |

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/Hayder-IRAQ/pdf-editor-pro.git
cd pdf-editor-pro
```

### 2. Create a virtual environment (recommended)
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Run
```bash
python main.py
```

---

## 📦 Dependencies

| Package | Purpose |
|---|---|
| `PyMuPDF` | PDF rendering and editing engine |
| `Pillow` | Image processing for page display |
| `tkinter` | GUI framework *(built into Python)* |

Install:
```bash
pip install PyMuPDF Pillow
```
